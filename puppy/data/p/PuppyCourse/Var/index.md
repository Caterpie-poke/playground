# 変数 

変数は、何かしらの値が与えられた式(expression)です。

まず、値を与えて変数を定義します。

```
x = 1    # 値を与えて変数を宣言する
```

一旦、定義した変数は数式の一部として自由に利用できます。

```
print(x)
print(x+1)
```
